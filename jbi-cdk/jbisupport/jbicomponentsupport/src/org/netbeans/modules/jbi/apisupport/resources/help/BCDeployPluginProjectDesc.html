Creates a new Netbeans plug-in project for Binding Component(BC) deployment plug-in in IDE. You can build, install and test the BC deployment plug-in using this project. It generates Netbeans plug-in artifacts and the WSDL extension schema file to add this BC specific configurations in a WSDL file using WSDL Editor. Also adds this BC component description to the layer.xml to create a service unit targeted to this BC in a service assembly in Composite Applications project.
