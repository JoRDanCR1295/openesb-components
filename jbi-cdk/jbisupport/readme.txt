jbisupport module suite builds set of modules related to Jbi Component development toolkit (JBI CDK). 

The following directory structure shows the module directories under the jbisupport suite.
jbicdk
    |-- ....
    |-- ....
    |-- jbisupport  = module suite to build the set of modules for the CDTK
          |--jbilib  =  jbi api library module
          |--jbicomponentsupport  =   jbi component projects module
          |--jbicomponent  = jbi component code generation templates project
          |--help = CDTK javahelp
          |--...
